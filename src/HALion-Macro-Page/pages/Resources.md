# Resources

Resources are the most basic objects and are used by controls to display graphics, animations, or text. The following types of resources are available: 

* [Bitmap](./Bitmap.md)
* [Color](./Color.md)
* [Decor (Resource)](./Decor-Resource.md)
* [Font](./Font.md)
* [Section](./Section.md)
* [SVG](./SVG.md)
